# cr-rb-c-benchmark
Benchmark for Tower of Hanoi algorithm using C, Ruby, Crystal

## Usage

### Required components
* UNIX like os
* gcc 7.3.0
* ruby 2.5.0
* crystal 0.21.0

### Launch
```sh
./run.sh
```
